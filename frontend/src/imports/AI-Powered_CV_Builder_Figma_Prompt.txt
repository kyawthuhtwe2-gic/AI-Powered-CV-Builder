## 📄 AI-Powered CV Builder - Figma Prompt (Premium SaaS UI)

Design a modern, premium SaaS web application called **“AI-Powered CV Builder”**.
The product allows users to create professional resumes using AI, dynamic forms, real-time preview, profile image upload, and **fully distinct CV templates with unique visual identities**.

---

# 🌐 GLOBAL HEADER (ALL AUTHENTICATED PAGES)

* **Left Side**

  * App Logo (icon / brand mark)
  * Title: **AI-Powered CV Builder**

* **Right Side**

  * Theme Toggle (Light / Dark)
  * User Account (Google Email Avatar + Dropdown)
  * Logout Button (icon or text)

---

# 🔁 CORE USER FLOW

1. User signs in via **Google OAuth**
2. Redirect to Dashboard
3. Click **“Create CV”**
4. Go to Template Selection Page
5. Select a CV Template (VISUALLY UNIQUE DESIGNS):

   * Modern (Tech / clean UI style)
   * Minimal (Ultra clean, lots of whitespace)
   * Corporate (Formal, structured grid, navy tones)
   * Creative (Bold typography, colorful accents, modern layout blocks)
6. Proceed to CV Builder Form
7. Fill CV data (auto-save enabled)
8. Live preview updates in real-time based on selected template design
9. Export or download as PDF
